# 🧩 Aplicación de Consola - Lista de Tareas (To-Do List)

Esta es una aplicación de consola desarrollada en **C# (.NET)** que permite gestionar una lista de tareas.  
El usuario puede **agregar, listar, marcar como completadas y eliminar tareas** fácilmente desde la terminal.

---

## 🚀 Características

- ➕ Agregar nuevas tareas con descripción y fecha límite (opcional).  
- 📋 Listar todas las tareas registradas.  
- ✅ Marcar tareas como completadas.  
- ❌ Eliminar tareas de la lista.  
- 💾 Gestión de datos usando una lista genérica (`List<T>`).  
- 🧱 Uso de clases, objetos y encapsulación.

---

## 🧰 Requisitos previos

Asegúrate de tener instalado:

- [.NET SDK 8.0 o superior](https://dotnet.microsoft.com/en-us/download)
- Un editor de código como [Visual Studio Code](https://code.visualstudio.com/) o [Visual Studio](https://visualstudio.microsoft.com/)

Verifica tu versión de .NET con el siguiente comando:

```bash
dotnet --version
```
